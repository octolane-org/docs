---
title: "Migrating from HubSpot"
description: "Step-by-step switch from HubSpot."
---

1. Export Companies, Contacts, Deals, and Engagements from HubSpot as CSV.
2. Connect Gmail and Calendar to Octolane.
3. Run [CSV import](/managing-data/importing) for each file. Mappings auto-suggest.
4. Connect Zapier or the API to forward any HubSpot-bound integrations.
5. Run side-by-side for one week, then cut over.

Need a hand? Email [help@octolane.com](mailto:help@octolane.com). We'll do the migration with you.
