---
title: "Auto-filling fields"
description: "Watching AI fields update themselves."
---

Define an [AI field](/introductions/ai-fields). Octolane reads every email, meeting, and note on the record and keeps the value current.

Updates appear with a small sparkle icon. Hover to see the source citation. Which message, which sentence. Edit manually anytime; manual edits lock the field until you unlock it.
