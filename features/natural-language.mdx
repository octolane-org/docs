# Create with natural language

Tell Octolane what happened in plain English; we create or update the record for you.

<CardGroup>
  <Card title="Examples" icon="message-plus">
    - “Create opp for Acme, $45k ARR, next step: pricing review next week.”  
    - “Log meeting with ZenCorp today, mark as stage: discovery.”  
    - “Add contact Priya @ Zephyr as champion.”
  </Card>
  <Card title="What we do" icon="magic-wand">
    Parse intent, map to the right account/contact, set fields, and attach the underlying email or meeting for traceability.
  </Card>
</CardGroup>

<AccordionGroup>
  <Accordion title="Accuracy guardrails" icon="shield-check">
    We validate owner, stage, and amount against existing records before writing to avoid duplicates.
  </Accordion>
  <Accordion title="Where to use it" icon="cursor-click">
    Use natural language in the web app or directly from email summaries we surface after a call.
  </Accordion>
</AccordionGroup>
