Manage your Octolane workspace from **Settings** in the sidebar. Here's what you can configure.

## General

- **Workspace name** - your team's workspace name, displayed in the sidebar and emails
- **Workspace URL** - your unique Octolane URL
- **Company details** - company name, website, industry, and size (used by agents for context)

## Agent settings

- **Agent behavior** - control which agents are active and configure their rules
- **Approval settings** - set which actions require approval and which can run automatically
- **Communication style** - define your team's tone and voice so agents match it

## Data model

- **Custom fields** - add, edit, or remove fields on deals, contacts, and companies
- **Custom objects** - create new object types beyond the defaults
- **Pipeline stages** - customize your sales pipeline stages

## Integrations

- **Connected accounts** - manage Gmail, Calendar, LinkedIn, WhatsApp, and other connections
- **Signal** - manage the Signal tracking script and configuration
- **Zapier** - manage your Zapier API key and connected automations
