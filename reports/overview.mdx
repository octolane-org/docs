---
title: "Reporting in Octolane"
description: "Reporting is conversational. Ask AI Chat for any report and it generates it live."
---

There is no separate report builder. Open AI Chat and ask:

> "Pipeline by stage, this quarter, grouped by rep, exclude churn."

The report renders inline as a table or chart. Save it as a view, pin to a dashboard, or schedule it to Slack/email.

See [Common report recipes](/reports/recipes).
